# AI プログラミング演習プロジェクト

このプロジェクトは、AI を活用したアジャイル開発のケーススタディを通じて、ソフトウェア開発の実践的なスキルを習得することを目的としています。

## プロジェクト概要

本プロジェクトでは、会議室予約システムを題材として、以下の内容を学習・実践します：

- 要件定義・分析から運用まで一貫した開発ライフサイクル
- アジャイル開発手法（エクストリームプログラミング）の実践
- テスト駆動開発（TDD）による高品質なソフトウェア開発
- ドメイン駆動設計（DDD）によるビジネスロジックの設計
- ヘキサゴナルアーキテクチャによる保守しやすいシステム構築

## 学習の進め方

1. **要件定義・分析** - ユーザーのニーズとシステムの目的を理解する
2. **設計** - システムの全体像と詳細設計を策定する
3. **開発** - TDD による実装とテストを繰り返す
4. **運用** - システムの配置・運用・保守を行う

各段階で参考資料を活用し、理論と実践を組み合わせながら学習を進めてください。
