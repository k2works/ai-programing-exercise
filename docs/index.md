# 実践 AIプログラミング プロジェクト概要

## プロジェクトの目的
AIプログラミングの実践的な学習と開発を通じて、よいソフトウェアの規律と知見を身につけることを目的とします。

## 技術スタック
- Go, JavaScript, Docker, MkDocs, PlantUML, Mermaid
- MCP Serverによる外部サービス連携（GitHub, Notion, Slack, Atlassian, Wiki.js）

## 機能概要
- FizzBuzzアプリケーションの実装
- テスト駆動開発（TDD）による品質向上
- ドキュメント自動生成・管理
- 開発日誌・Wiki・Issue管理の自動化

## システム構成
- `app/`: アプリケーション本体（Go言語）
- `docs/`: ドキュメント・日誌・Wiki
- `scripts/`: ドキュメント生成・管理スクリプト
- `README.md`: プロジェクト概要・セットアップ手順
- `mkdocs.yml`: ドキュメント生成設定
- `package.json`: 依存関係・スクリプト
